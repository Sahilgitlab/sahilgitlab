# 📦 Python Library

## VectorQuant

A Python library for **Vector Quantization and clustering** designed for machine learning pipelines.

### Install

```bash
pip install vectorquant
```

### Example

```python
from vectorquant import VectorQuant

vq = VectorQuant(k=256)

vq.fit(data)

compressed = vq.encode(data)

decoded = vq.decode(compressed)
```
